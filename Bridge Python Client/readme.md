# Running the Tests
- Install the project dependencies with the command: `pip install -f requirements.txt`. 
- You could install the dependencies in a virtual environment [virtual environment](https://docs.python.org/3/library/venv.html)
- create a .env file with a variable which points to where your .m2: `PATH_TO_M2=path/to/m2 parent directory/`

NB: The Java project should be running on the default Py4J port of 25333 to successfully connect to the JVM
